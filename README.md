# PayZen for X-Cart

PayZen for X-Cart is an open source plugin that links e-commerce websites based on X-Cart to PayZen secured payment gateway developped by [Lyra Network](https://www.lyra-network.com/).

We will put the plugin source code in this repository shortly.
